# Enterprise Microsoft 365 Administration Lab

## Project Overview

This project simulates the design and administration of a Microsoft 365 environment for a 250-user telecommunications and technology organization.

The objective is to demonstrate practical understanding of enterprise identity, security, collaboration, device management, governance, monitoring, and automation using Microsoft 365 services.

## Organization

**Organization:** Osegbo Technologies Limited  
**Industry:** Telecommunications and Technology  
**Number of Users:** 250  
**Primary Location:** Abuja, Nigeria  
**Regional Offices:** Lagos and Enugu

## Project Objectives

The project aims to:

- Design an enterprise Microsoft 365 environment.
- Implement identity and access management principles.
- Apply security controls and Zero Trust principles.
- Design device management policies.
- Configure collaboration services.
- Develop governance and monitoring processes.
- Demonstrate PowerShell automation.
- Document enterprise implementation decisions.

## Scope

The project covers:

- Microsoft Entra ID
- Microsoft 365 administration
- Identity and access management
- Conditional Access
- Multi-factor authentication
- Microsoft Intune
- Microsoft Defender
- Exchange Online
- Microsoft Teams
- SharePoint Online
- PowerShell automation
- Governance and monitoring

## Expected Outcomes

At the end of the project, the repository will contain:

- Enterprise architecture documentation.
- Security and identity design.
- Implementation guides.
- Architecture diagrams.
- PowerShell automation scripts.
- Testing and validation documentation.
- Lessons learned.
- Recommendations for future improvements.

## Disclaimer

This is a simulated laboratory environment created for educational and professional portfolio purposes. No confidential organizational data, credentials, or production information is included.
